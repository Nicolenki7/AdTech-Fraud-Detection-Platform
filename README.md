## 📋 Overview

Complete AdTech fraud detection platform with real-time scoring, investigation dashboard, and automated alerting. Integrates data ingestion, feature engineering, model scoring, and operational workflows.

---

## 💼 Business Impact

- **Real-Time Detection**: Immediate fraud identification
- **Investigation Workflow**: Prioritized suspect queue for analysts
- **Business Intelligence**: Dashboard for fraud trends and patterns
- **Cost Reduction**: Prevent fraudulent ad spend before payment

## 🚀 Key Features

### Real-Time Scoring
- API endpoint for fraud probability
- Sub-100ms response time

### Investigation Dashboard
- Prioritized suspect queue
- Manual review workflow
- Feedback loop for model improvement

### Alerting System
- Threshold-based alerts
- Email/Slack notifications
- Escalation rules

## 🔧 Setup & Installation

```bash
# Clone the repository
git clone https://github.com/Nicolenki7/AdTech-Fraud-Detection-Platform.git
cd AdTech-Fraud-Detection-Platform

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp config.py.example config.py

# Run platform
python src/main.py
```
